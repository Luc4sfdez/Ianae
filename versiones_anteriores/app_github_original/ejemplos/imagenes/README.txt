Estructura de directorio para imágenes de entrenamiento:

1. Las imágenes deben colocarse en este directorio (ejemplos/imagenes/)
2. Nombre recomendado para las imágenes:
   - horizontal_1.jpg, horizontal_2.jpg, etc. (para orientación 0°)
   - vertical_1.jpg, vertical_2.jpg, etc. (para orientación 90°)
   - horizontal_180_1.jpg, etc. (para orientación 180°)
   - vertical_270_1.jpg, etc. (para orientación 270°)

3. Formatos soportados (totalmente compatibles):
   - PNG (recomendado para calidad sin pérdida)
   - JPG (para fotografías)
   - BMP (sin compresión)
   - WEBP (formato moderno)

4. Tamaño recomendado:
   - Mínimo: 224x224 píxeles
   - Ideal: 500x500 píxeles o más

5. Ejemplo de estructura:
   ejemplos/imagenes/
   ├── horizontal_1.jpg
   ├── vertical_1.jpg  
   ├── horizontal_2.jpg
   └── vertical_2.jpg
